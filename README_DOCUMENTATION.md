## üìû Quick Reference

### Endpoints
```
GET  /api/facebook-status  - Check connection status
POST /api/facebook-login   - Login with email/password
POST /api/facebook-logout  - Logout and clear session
GET  /api/facebook-check   - Quick status check
```

### Functions
```
checkFacebookSessionStatus()      - Check session on page load
handleFacebookLogin()             - Process login form
logoutFromFacebook()              - Handle logout
showToastNotification()           - Display notifications
```

### Configuration
```
Session duration: 7 days
Session type: Filesystem
Cookie security: HTTPONLY + SAMESITE=Lax
Auto-refresh: On each request
```
